## 🚀 To Get Running

### Kill Existing Processes
```powershell
Get-Process node | Stop-Process -Force -ErrorAction SilentlyContinue
```

### Start Dev Server
```bash
cd d:\flyfree\flyfree-platform
npm run dev
```

**Wait for:**
```
✓ @flyfree/web:dev    Ready
✓ @flyfree/api:dev    Started  
✓ @flyfree/admin:dev  Ready
```

### Access Everything
```
User App:    http://localhost:3000
Admin:       http://localhost:3002
API Docs:    http://localhost:3001/docs
Server Logs: http://localhost:3001/api/admin/logs
```
